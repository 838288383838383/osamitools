# osamitools

Scoop bucket for osamito's tools.

## Install

```bash
scoop bucket add osamitools https://github.com/838288383838383/osamitools
scoop install osamitools/catdot
```

## Available Apps

| App | Description |
|-----|-------------|
| [catdot](bucket/catdot.json) | CatDot Engine - A modern game engine with GUI editor, Python/Gleam/Nim scripting, Discord integration, Godot import, and more |
